Exercices Algorithmie en Javascript

Test techniques Javascript, pas besoin de cloner ce projet La page d'accueil de votre site doit inclure un lien vers les pages dédiées à chaque exercice.

Hébergez votre projet sur GitHub ou GitLab, selon votre préférence.

Pour chaque exercice, créez une page HTML spécifique accompagnée d’un fichier JavaScript unique.

Les exercices doivent être réalisés uniquement dans la console du navigateur. Assurez-vous que les résultats s’affichent via un console.log.


Documentation technique :
Dans chaque page HTML, listez brièvement les notions ou méthodes JavaScript utilisées pour résoudre l’exercice.
Après avoir terminé chaque exercice, effectuez un commit et un push de vos modifications sur le dépôt Git.
